What does "Full" in Full Stack mean?

	The "Full" in Full Stack means that all software creation tasks are in the front, 
	back, and in between.
	
	Front end is what the client(s) would see, back end is the server side
	of the front end.
	
	(click here for a visual)

What does "Stack" in Full Stack mean?

	The "Stack" in Full Stack is a computer lingo for a group of programming languages
	and utilities that supports software development.
	
	The front end would have a HTML5, CSS3, and Javascript.
	The back end would have MySQL and PHP.
	
	(click here for a visual)

What does "LAMP STACK" mean?

	LAMP is short for 
	
	Linux
	Apache
	MySQL
	PHP
	
	It's an open-source Web development
	platform, also called a Web stack. It uses Linux as the operating system, Apache as
	the Web server, MySQL as the RDBMS, and PHP as the object oriented scripting language.

1. Explain database: A database is a data organizer that helps businesses and companies organize information in servers
Ex: dentists, repair shops, etc.

2. Explain MySQL: MySQL is a database, which is ranked second in the most-used databases. Developed by Oracle, which is ranked first in the most-used databases. MySQL is free while Oracle has paid service.
Fun Fact: The creator named it "MySQL" after his daughter, My.

3. Explain SQL: "Structured Query Language" -- MySQl is a name for the database, while SQL is the LANGUAGE of this database used to answer questions such as: "which products get bought a lot and which isn't?" It's an international standard language that can work with MySQL, Oracle SQL, Microsoft SQL server, and many others!

4. Explain PHP: PHP is a programming language that can connect the front end and back end together. It can read HTML forms then sends that form info into the server. It can also read the database, and sends that back to another HTML form. PHP is a server-side language, which is totally secure and private -- you can't really see PHP when you do a "view source".

5. MySQL Table: It has rows(horizontal) and columns(vertical), where data is organized in cells. It looks like an Excel spreadsheet, but you'll realize that it's actually different. (find the perfect table on Google)
